# 2주차 Overview

## 학습 목표
- 옵저버 패턴으로 상태 변화 알림과 구독 해제를 구현한다.
- `Stream` 기반 이벤트 브로드캐스트를 Flutter/Dart에서 실습한다.
- 단위 테스트로 지연 전파와 에러 전파 시나리오를 검증한다.

## 준비 사항
- `lib/src/observer/`에 핵심 예제와 Riverpod 연동 샘플 작성
- `test/observer/`에서 스트림 이벤트, 구독 취소, 에러 흐름 테스트
- 위젯 예제에서 UI 갱신과 스낵바 알림을 연동하여 행동 관찰
