# 3주차 Overview

## 학습 목표
- 데코레이터 패턴으로 위젯/서비스 기능을 동적으로 확장한다.
- 컴포지트 패턴으로 계층형 데이터와 UI 구성을 관리한다.
- 다단계 스타일링과 비용 측정을 통해 리빌드 최적화를 학습한다.

## 준비 사항
- `lib/src/decorator/`와 `lib/src/composite/`에 콘솔·위젯 예제 배치
- 데코레이터 체인 구성 및 성능 로그를 남기는 유틸 작성
- `test/decorator/`, `test/composite/`에서 happy/edge 케이스 커버

## 산출물 체크리스트
- 리빌드 횟수를 기록하는 믹스인 또는 헬퍼 제공
- README 10줄 요약, 안티패턴 비교 5줄, 과제 3개
- UML 텍스트 다이어그램과 Riverpod provider 관계 표
